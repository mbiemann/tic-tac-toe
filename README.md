## Contributions

Pre-requirements:

* [NVM](https://github.com/nvm-sh/nvm)

```shell
nvm use $(cat .node-version)
npm install
```

Run:

```shell
npm start
```

Build:

```shell
npm run build
```

Run a sandbox script:

```shell
node sandbox/sandbox.js
```
